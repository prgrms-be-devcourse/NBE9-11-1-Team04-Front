## 📁 .env 파일 설정 방법

프로젝트 루트 디렉토리에 `.env` 파일을 생성한 후, 아래 형식에 맞게 환경 변수를 작성해주세요.

```env
# 백엔드 주소 작성
ex. NEXT_PUBLIC_BASEURL = "{백엔드 서버 주소}/api/v1"
```

### ⚠️ 주의사항
- `.env` 파일은 **절대 Git에 커밋하지 마세요** (`.gitignore`에 추가 필수)
- 값(value)에는 따옴표를 사용하지 않습니다 (필요한 경우 제외)
- 공백 없이 `KEY=VALUE` 형식을 유지해야 합니다
